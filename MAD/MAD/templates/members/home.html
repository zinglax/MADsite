Mobile App Developers Club

Welcome to the home site of the University of Maryland Mobile Application Developers Club. Use the Navigation bar in the head banner to browse the site. This site is created with Django, a python web framework, and jQuery Mobile in order to render well on all device browsers. We try to develop applications that are supported by all mobile platforms. Our goal is to motivate UMD students about developing for mobile platforms and create applications that will benefit students on campus.
